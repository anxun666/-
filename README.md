----------
简介：
这是一个普普通通的一个php开源跳转网页，支持WX🛰️和QQ🐧的网页访问跳转至浏览器，简洁好用，多端支持（Windows、Android、iOS、鸿蒙、塞班等系统）
----------------
使用教程（简）：
1.新建文件夹

2.上传插件整个文件到jump目录

2.修改你网站首页文件。

3.在网页首页文件的文件第一行<?php 下新增一行：require_once('./jump/WxqqJump.php'); 
文件位置自行修改即可，不然可能会报错哦！
*详见使用说明教程.txt
-------------------
温馨提示：
1.github可直接克隆仓库至服务器，也可直接下载main等
2.第5步骤请重视，谨慎设置避免出错
3.当不再使用或者需要临时关闭跳转时，只需//注销该行代码即可
4.所有文件均可修改部分内容，修改不当可能会报错或网页崩溃
